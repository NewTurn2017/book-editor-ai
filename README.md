# 📚 AI 출판 기획 질문지

간단하고 예쁜 Q&A UI 애플리케이션으로 출판 기획을 위한 질문-답변 시스템과 AI 분석 기능을 제공합니다.

## ✨ 주요 기능

- 🎨 **아기자기한 파스텔 디자인**: 핑크, 민트, 라벤더 컬러의 부드러운 UI
- 📝 **10개 질문**: 3개 카테고리로 구성된 출판 기획 질문
- 📊 **실시간 진행률**: 답변 진행 상황 실시간 표시
- 🤖 **AI 종합 분석**: Claude AI 기반 맞춤형 출판 기획 조언
- 💾 **자유 입력**: 각 질문에 추가 의견 작성 가능

## 🚀 사용 방법

1. 웹사이트 접속
2. 10개 질문에 순서대로 답변
3. 진행률 100% 도달 시 "AI 종합 분석 받기" 버튼 클릭
4. AI 분석 결과 확인

## 🛠️ 기술 스택

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **AI**: Claude API (Anthropic)
- **Design**: 파스텔 컬러 테마, 그라데이션, 애니메이션
- **Deployment**: Vercel

## 📋 분석 결과

AI가 다음 7가지 영역에 대해 종합 분석을 제공합니다:

1. 📚 당신의 책 프로필
2. 🎯 핵심 메시지
3. ✨ 추천 제목 아이디어 (5개)
4. 👥 타겟 독자층 분석
5. 🎨 추천 문체 및 톤
6. 📖 유사 도서 스타일
7. 💡 다음 단계 추천 (5개)
